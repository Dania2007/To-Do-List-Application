# To-Do-List-Application
To-Do List Application

Description:
This is a Java console-based application used to manage daily tasks.
Users can add, view, complete, and delete tasks.

Features:
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks

Technologies Used:
- Java
- OOP concepts
- ArrayList
- Exception Handling

How to Run:
1. Open the project in a Java IDE
2. Run ToDoListApp.java
3. Use the menu options

Project Files:
- ToDoListApp.java
- InvalidTaskException.java
